Bem vindo ao mundo maravilhoso da web, (X)HTML e CSS. Com apenas um pouco de conhecimanto, alguma prática e algo a dizer, você pode criar seu próprio pedacinho do ciberespaço ou melhorar um trabalho existente.

Você percebera que usamos (X)HTMLpor todo este livro. Ele é um acrônimo que significa tanto para HTML quanto para XHTML, sendo que HTML significa Hypertext Markup Language e XHTML Extensible Hypertext Markup Language. Anda que HTML e XHTML não sejam idênticos, eles são suficientemente parecidos para que essa referência faça sentido.

Este livro é seu guia realista para compreender documentos web, enfeitar uma página existente ou criar página complexas e animadas que integram designs, multimídias e criação de script.

A melhor maneira de começar a trabalhar com o HTML é entrar nele, portanto, é isso que este capítulo faz: ele te aprofundar-se nos fundamentos básico de como (X)HTML e CSS trabalham por trás dos bastidores dentro de páginas web, introduzindo-o em suas estruturas fundamentais. Quando terminar este capítulo, você saberá como o (X)HTML e CSS funcionam, de forma que poderá começar a criar ou editar páginas web imediatamente.
